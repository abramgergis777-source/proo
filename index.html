🔐 Digital Safe Lock System

Digital Logic Design Project

A complete digital safe lock system designed using fundamental digital logic components and logic gates. The system is based on a 4-digit binary password with intelligent access control.

A correct password grants full access to the system. An incorrect entry increments an error counter, and after 3 consecutive wrong attempts, the system triggers a full lock and stops responding until manually reset.

Key Features
XOR Comparison Logic Counters & Lock Mechanism Even / Odd Detection (LSB) Seven Segment Displays Password Hiding
